# Daily Horoscope Website using Chat GPT API

## Technologies Used

- HTML
- CSS
- JavaScript
- jQuery
- Bootstrap
- Chat GPT API

## Installation and Setup

1. Clone the repository to your local machine
2. Open the index.html file in your browser
3. Select your zodiac sign from the dropdown menu
4. Click on the "Get Horoscope" button to generate your horoscope prediction

## Features

- Daily horoscope prediction for each zodiac sign
- Personalized horoscope readings based on user's zodiac sign
- User-friendly interface
- Easy to use and navigate
- Mobile responsive design

## Future Improvements

- Daily notification to users about their horoscope prediction
- More personalized and accurate horoscope readings using machine learning algorithms
- Integration with social media platforms for easy sharing of horoscope predictions
- Option for users to receive horoscope predictions via email or SMS
